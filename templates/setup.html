Create your (free) Azure app registration — ~3 minutes, one time

  1. Go to portal.azure.com and sign in with any Microsoft account.
  2. Search for App registrations → New registration.
  3. Name it anything (e.g. Halo Stats); under Supported account types choose Personal Microsoft accounts (or "any org + personal").
  4. Under Redirect URI pick platform Mobile and desktop applications (add it later via Authentication → Add a platform if you missed it) and enter {{ redirect_uri }}.
  5. Click Register, then copy the Application (client) ID from the Overview tab into the field below.
  6. Go to Certificates & secrets → New client secret, create one, and copy its Value (not the Secret ID!) into the field below.
